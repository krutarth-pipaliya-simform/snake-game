# Branch Naming

## Format
```
<type>/<area>/phase<N>-<short-slug>
```

- `area`: `fe`, `be`, or `shared`
- `N`: phase number (0-11 for FE, 1-6 for BE). Use `misc` for non-phase work.

## Examples
```
feat/fe/phase1-snake-movement
feat/be/phase2-lobby-events
feat/shared/extract-types
chore/repo/husky-setup
fix/be/phase3-collision-bug
```
